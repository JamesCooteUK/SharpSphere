VMware vSphere Management SDK Readme

The vSphere Management SDK is a bundle that contains a set of VMware vSphere SDKs. Each SDK supports the development of management applications that call VMware API functions to access vSphere servers. The following table lists the different SDKs and provides links to documentation.
